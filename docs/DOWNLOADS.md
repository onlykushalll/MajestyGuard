# Downloads

Official MajestyGuard downloads will be published through GitHub Releases:

https://github.com/onlykushalll/MajestyGuard/releases

MajestyGuard intends to use the SignPath Foundation for code signing via
SignPath.io's free open-source signing program. Until that signing path is
approved and a signed release is published, MajestyGuard should be treated as an
active research prototype rather than a production-ready login replacement.

## Current Status

There are no production-ready signed downloads yet.

For now, reviewers and contributors should inspect the source repository:

https://github.com/onlykushalll/MajestyGuard
